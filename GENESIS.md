# GENESIS

GENESIS is the supreme law for METAOS.

Human defines only:
- goal
- essence
- constraints
- acceptance

System performs:
- signal interpretation
- generation
- evaluation
- selection
- mutation
- archival
- repetition
- policy evolution
- domain expansion

Canonical loop:
- `signal -> generate -> evaluate -> select -> mutate -> archive -> repeat`

Operational state law:
- append-only truth is the sole authority
- replay reconstructs effective state from truth
- `civilization_state` is derived operational state, never an alternate canonical loop

Invariants:
- append-only truth
- replayable effective state
- immutable artifacts
- minimal core
- domain autonomy
- lineage ecology
- pressure-coupled exploration

Artifact law:
- everything that evolves is an artifact
- every mutation creates a new immutable artifact
- lineage remains replayable

Pressure law:
- novelty
- diversity
- efficiency
- repair
- domain_shift
- reframing

Failure protocol:
- `plateau -> exploration_collapse -> diversity_repair_failure -> repair_escalation -> invalid_state -> replay_restore`

Ultimate law:
- human defines goals
- system evolves solutions
