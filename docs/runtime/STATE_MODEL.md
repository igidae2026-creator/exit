# State Model

## Primary Truth

The only durable truth is append-only event, metric, registry, signal, and archive history.

## Effective State

`civilization_state` is a replayed effective state derived from:

- replayed append-only truth
- recent memory windows
- domain lifecycle state
- lineage ecology state
- economy and stability summaries

## Derived Caches

- checkpoints and archive summaries are acceleration layers only
- they may speed restart, but they do not replace replay authority

## Required Fields

- lineage counts and diversity
- domain lifecycle buckets
- policy and evaluation populations
- stability and economy scores
- append-only, invariant, and replay mismatch violation lists
